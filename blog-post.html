CSS Grid in 10 Minutes – A Beginner's Guide

By Jason Carter  |  📅 March 12, 2026

CSS Grid is one of my favorite things in CSS. When I first saw it I thought it looked complicated, but once I started using it I realized it was actually really simple. Let me show you the basics.

What is CSS Grid?

CSS Grid is a layout system that works in two directions at once — rows and columns. Unlike Flexbox which is mostly one direction, Grid lets you control both at the same time. Think of it like a spreadsheet for your webpage.

Setting Up a Grid

All you need to do is add display: grid to a parent element. Then you use grid-template-columns to say how many columns you want. 

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */
  gap: 20px; /* space between the items */
}

The fr unit means "fraction of the available space". So 1fr 1fr 1fr just means three equal columns. That's it!

The Auto-Fill Trick (This is the one I use most)

This one line of CSS makes a responsive grid with no media queries: 

.grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 20px;
}

This creates as many columns as will fit on screen, and each one is at least 250px wide. On a phone you get 1 column. On a tablet maybe 2. On desktop maybe 3 or 4. No breakpoints needed! This is how I made the project cards on my portfolio.

Placing Items at Specific Spots

You can also tell a specific item exactly where to sit using grid-column: 

.big-item {
  grid-column: 1 / 3; /* starts at column line 1, ends at line 3 (spans 2 columns) */
}

Quick Summary

  • Add display: grid to the parent
  • Use grid-template-columns to define your columns
  • Use gap for the space between items
  • Use auto-fill + minmax for responsive layouts
  • Use grid-column to place items manually

My advice: just go build a card grid right now. That's how CSS Grid clicked for me. Don't just read about it — type it out and play with the numbers!
